## 🎯 Project Overview

A sophisticated, high-end philanthropic organization website inspired by Alwaleed Philanthropies. The platform features:

- **Premium Professional Design** with deep navy blue, gold accents, and clean white spaces
- **Responsive & Mobile-First** layout
- **Multi-Language Support** (Arabic & English)
- **RTL (Right-to-Left)** support for Arabic
- **Comprehensive Grant Application Portal** with 5-step form
- **WhatsApp Integration** for direct communication
- **Video & Media Support**
- **Smooth Animations & Transitions**

### English Description
A professional philanthropic website with a sophisticated 5-step grant application form, bilingual support (Arabic/English), responsive design, and WhatsApp integration for applicant communication.

### Arabic Description
موقع مؤسسة خيرية احترافي مع نموذج طلب منحة متقدم من 5 خطوات، دعم ثنائي اللغة (عربي/إنجليزي)، تصميم سريع الاستجابة، وتكامل واتس آب للتواصل مع المتقدمين.

### 📱 Application Portal

#### **5-Step Multi-Step Form**

**Step 1: Personal Information**
- Full Name (Arabic)
- Country of Residence (20+ Arab countries)
- Marital Status dropdown
- Number of Children

**Step 2: Contact & Career**
- Phone Number (auto-formatted)
- Email Address
- Job/Profession
- Monthly Income (formatted)

**Step 3: Grant Details**
- Grant Type (10 categories):
  - Housing
  - Investment
  - Small Projects
  - Debts
  - Marriage
  - Education
  - Car
  - Needy Assistance
  - Sickness
  - Donation
- Amount Requested
- Detailed Description (textarea)

**Step 4: Banking Information**
- Bank Name (13+ Saudi banks)
- Account Holder Name
- IBAN Number (auto-formatted)
- Security information box

**Step 5: Attachments & Submission**
- ID Card Front (drag & drop)
- ID Card Back (drag & drop)
- File preview
- Terms & Conditions checkbox
- Submission button

#### **Advanced Features**
✅ Progress bar with visual indicators
✅ Step validation
✅ Auto-formatting for:
   - Phone numbers (+966 XX XXXX XXXX)
   - IBAN (SA00 0000 0000 0000 0000 0000)
✅ File upload with drag & drop
✅ File size validation (max 5MB)
✅ File type validation (JPG, PNG, PDF)
✅ Real-time form validation
✅ Success modal with Transaction Number
✅ Automatic WhatsApp redirect with pre-filled message
